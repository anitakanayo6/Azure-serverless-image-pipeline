# Azure-serverless-image-pipeline


## Overview

This project is a cloud-native, event-driven system built using Microsoft Azure. It automatically processes images uploaded to cloud storage using serverless computing.

## Goal

To demonstrate real-world cloud engineering concepts including:

* Event-driven architecture
* Serverless computing
* Cloud storage
* Automation workflows
* Monitoring and logging

## Architecture

Images uploaded to Azure Blob Storage automatically trigger Azure Functions, which process the images and generate thumbnails.

## Azure Services Used

* Azure Blob Storage
* Azure Functions
* Azure Monitor
* Application Insights
